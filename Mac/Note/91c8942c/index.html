Mac Note：Mac 常用技巧

發表於 2023-01-06 更新於 2024-04-05 分類於 Mac ， Note 閱讀次數： Disqus： 文章字數： 515 所需閱讀時間 ≈ 2 分鐘

列出我日常使用 mac 時遇到的一些障礙。

Mac 鍵盤快速鍵

Mac 鍵盤快速鍵
【Mac OS 教學】33 個 Mac 鍵盤要學會的快速鍵！（更新）
直接進設定看最準

切換輸入法

ctrl + 空白鍵
caps

重做

command + shift + Z
不太確定什麼情況可以觸發

Mission Control

ctrl + 上鍵

顯示桌面

fn + F11
觸控板大拇指與三隻手指外展

再做一次可以回到原本的視窗
視窗全螢幕下無效

切換桌面

ctrl + 右鍵
ctrl + 左鍵

切換開啟的應用程式

command + tab
command + shift + tab

調整顯示大小

放大：command + =
縮小：command + -
預設：command + 0

截圖

擷取全螢幕畫面並儲存：command + shift + 3
自行選取截圖範圍並儲存：command + shift + 4
擷取特定視窗並儲存：command + shift + 4 + 空白鍵
截圖與錄影選項：command + shift + 5

瀏覽器

重新整理：command + R
捲到最上方：command + 上鍵
捲到最下方：command + 下鍵
開新視窗：command + N
開新分頁：command + T
重新開啟一個關閉的分頁：command + shift + T
關閉分頁：command + W

關閉當前程式

command + Q
